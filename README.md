## 配置
进入`config.json`文件配置，示例如下：
```
{   
    //日期设置
    "targetDate": {
        "year": 2023,
        "month": 4,
        "day": 24,
        "hours": 0,
        "minutes": 0,
        "seconds": 0,
        "milliseconds": 0
    },
    "texts": [
        "世间最好的默契",
        "并非有人懂你的言外之意",
        "而是有人懂你的欲言又止",
        "希望有人懂你的低头不语",
        "小心翼翼守护你的孩子气",
        "愿你找到一个像太阳一样的人",
        "为你晒掉所有不值一提的悲伤"
    ]
}
```
